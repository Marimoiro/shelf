これは何

カバンの商品カタログです。

操作方法

棚の回転方法

棚を回転させる方法は3種類あります。

  1. 棚の左右のアイコンをクリックする。
  2. 画面下のバーの数字をクリックする。
  3. 棚をドラッグする。

商品の詳細情報

カバンのパネルをクリックすると、パネルが手前にポップアップし、商品名や値段などの詳細情報を見ることができます。

検索機能

画面上部のナビゲーションバーに、「id」「price」「color」などの項目があります。

これらの項目をクリックすることで、それぞれ、商品ID、値段、色によってカバンをソートできます。

もちろん、昇順ソートと降順ソートの両方に対応しています。

設定

画面上部のナビゲーションバーの歯車のようなアイコンを開くと設定画面が開きます。

ここから、次の項目を設定出来ます。

  1. アニメーションの動きにEaseOutを効かせるかどうか。
  2. ステータスバーのタイプ。

開発情報

開発には次のようなものを使いました。
  1. MatrixEngine (WebGL)
  2. JavaScript (jQuery)
  3. HTML5, CSS3
  4. Bootstrap
